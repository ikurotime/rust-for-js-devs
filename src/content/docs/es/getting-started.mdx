# Primeros Pasos con Rust

¡Bienvenido a Rust para Desarrolladores de JavaScript! Esta guía te ayudará a hacer la transición de JavaScript a Rust partiendo de tus conocimientos existentes. Exploraremos los conceptos de Rust desde la perspectiva de JavaScript, destacando similitudes y diferencias en el camino.

## Cómo Está Estructurada Esta Guía

Esta guía sigue la estructura del libro The Rust Programming Language (a menudo llamado "El Libro de Rust"), pero pensando en los desarrolladores de JavaScript. Para cada tema:

1. Explicaremos el concepto de Rust
2. Mostraremos código JavaScript equivalente cuando sea aplicable
3. Destacaremos las principales diferencias y similitudes
4. Proporcionaremos ejemplos prácticos para reforzar el aprendizaje

## Lo Que Aprenderás

Siguiendo esta guía, aprenderás:

- Los fundamentos de la sintaxis y el sistema de tipos de Rust
- Cómo funciona el modelo de propiedad de Rust comparado con la recolección de basura de JavaScript
- Construir estructuras de datos con structs y enums
- Manejo de errores con los tipos Result y Option
- Crear código modular con paquetes, crates y módulos
- Trabajar con colecciones como vectores, strings y mapas hash
- Características avanzadas como genéricos, traits y concurrencia
- Construir aplicaciones del mundo real con Rust

## Prerrequisitos

Esta guía asume que tienes:

- Conocimiento práctico de JavaScript (ES6+)
- Familiaridad con conceptos de programación como variables, funciones y flujo de control
- Comprensión básica del uso de terminal/línea de comandos
