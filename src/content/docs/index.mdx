## Why Learn Rust as a JS Developer?

<CardGrid stagger>
  <Card title="Memory Safety" icon="shield">
    Rust's ownership system prevents memory-related bugs without a garbage collector.
  </Card>
  <Card title="Performance" icon="rocket">
    Get C-like speed with modern syntax and productivity similar to JavaScript.
  </Card>
  <Card title="Concurrency" icon="random">
    Write thread-safe code with Rust's ownership model and avoid race conditions.
  </Card>
  <Card title="WebAssembly" icon="puzzle">
    Rust is a first-class language for WebAssembly, enabling high-performance web applications.
  </Card>
</CardGrid>

## Course Structure

<CardGrid>
  <Card title="Getting Started" icon="seti:rust">
    Install Rust, write your first program, and learn the basics of Cargo - Rust's package manager.
  </Card>
  <Card title="Programming Concepts" icon="open-book">
    Learn about variables, types, functions, and control flow with JavaScript comparisons.
  </Card>
  <Card title="Ownership System" icon="key">
    Understand Rust's unique memory management through ownership, borrowing, and lifetimes.
  </Card>
  <Card title="Data Structures" icon="puzzle">
    Master structs, enums, and collections - the Rust equivalents of JavaScript objects and arrays.
  </Card>
  <Card title="Error Handling" icon="warning">
    Discover how Rust's Result and Option types provide safer alternatives to JavaScript exceptions.
  </Card>
  <Card title="Practical Projects" icon="laptop">
    Build real-world applications using Rust, from CLI tools to WebAssembly components.
  </Card>
</CardGrid>
